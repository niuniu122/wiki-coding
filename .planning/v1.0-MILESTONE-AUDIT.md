## Cross-Phase Findings

1. **Runtime to Vault/Wiki:** `finalize_runtime_session` has no production caller, and no production type implements `WikiGenerationPort`.
2. **Project discovery entry:** `ProjectDiscovery::discover` is correctly BM25-first and candidate-bounded, but `index projects search` requires `--catalog`; the normal agent and TUI do not reach it.
3. **Command outcomes:** parsing covers the locked surface, while `/retry`, `/provider`, `/api`, `/interrupt`, `/models`, and `/trace` have incomplete or substitute behavior.
4. **Distribution:** npm contains the launcher and legacy JavaScript but no native binary; native archives contain the launcher and binary but no legacy command.
5. **Cutover evidence:** the executable hosted-gate fixture records pre-cutover run `29474558013`; final run `29476499926` exists only in prose.
