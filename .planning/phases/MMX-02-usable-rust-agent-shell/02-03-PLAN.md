<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Files/env/CLI -> resolved config | Untrusted layered values choose endpoint, limits, and Provider identity |
| Environment/keyring -> HTTP client | Secret material crosses exactly once and must not reach logs/state |
| User text/runtime event -> terminal | Control characters and oversized output must not corrupt terminal state |
| CLI driver -> adapters | Effect ordering determines durable status, shutdown, and exit truthfulness |

</threat_model>
