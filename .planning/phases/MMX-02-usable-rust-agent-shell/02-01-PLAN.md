<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User/config/credential -> Provider request | Untrusted endpoint and secret shape outbound HTTPS |
| Provider bytes -> adapter | Remote untrusted streaming bytes may be malformed, huge, or secret-bearing |
| Adapter events -> core | Only normalized allowlisted events may affect durable runtime state |
| Core effects -> composition driver | Ordering determines whether completion/recovery claims are truthful |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Information Disclosure | Provider error/frames | high | mitigate | Fixed errors, bounded decoder, no raw response/body/reasoning retention |
| T-02-02 | Spoofing | endpoint/profile identity | high | mitigate | Validated pinned identity, HTTPS/non-loopback policy, redirects disabled |
| T-02-03 | Tampering | stream terminal order | high | mitigate | Existing strict reducer plus run-machine one-terminal invariant |
| T-02-04 | Denial of Service | unbounded stream/event | high | mitigate | Event/body bounds, deadline, cancellation, backpressure-ready API |
| T-02-05 | Elevation of Privilege | observed tool call | high | mitigate | Phase 2 records and terminates; no tool adapter invocation exists |
</threat_model>
