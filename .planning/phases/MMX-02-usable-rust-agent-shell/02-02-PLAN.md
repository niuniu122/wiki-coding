<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Process -> workspace lease | Competing or crashed processes must not become concurrent writers |
| Journal bytes -> replay | Local files may be truncated, tampered, oversized, or schema-incompatible |
| Session history -> compactor | Only completed visible context is eligible for future model context |
| Runtime facts -> trace/diagnostics | Secret-bearing or private data must be dropped before persistence/display |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-02-06 | Tampering | journal/index | critical | mitigate | strict schema/hash replay, atomic index, final-fragment-only repair |
| T-02-07 | Spoofing | workspace writer | high | mitigate | non-blocking exclusive OS lease held for process lifetime |
| T-02-08 | Information Disclosure | trace/compaction | high | mitigate | allowlists, bounds, redaction, completed-visible-only inputs |

| T-02-10 | Repudiation | retry/recovery | medium | mitigate | immutable terminal records, linked IDs, durable recovery receipts |
</threat_model>
