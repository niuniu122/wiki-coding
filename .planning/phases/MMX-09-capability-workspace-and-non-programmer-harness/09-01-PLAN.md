# Plan 09-01: Workspace, Card Schema, and Typed Retrieval

## Goal

Create the dedicated `capabilities/` source workspace, strict shared card contract, and compile-time isolated project/Skill/MCP lexical indexes.

## Tasks

1. Add catalog layout and boundary documentation with project data migrated into the new source workspace and empty valid Skill/MCP catalogs.
2. Add strict card/catalog validation, fingerprints, kind-specific documents, and typed markers.
3. Implement deterministic multi-kind BM25 recall plus candidate-only optional embedding rerank.
4. Add schema, isolation, fingerprint, lexical, and semantic failure tests.
