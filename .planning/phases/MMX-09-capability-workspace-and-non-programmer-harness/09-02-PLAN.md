# Plan 09-02: Readiness, CLI, and Authority Guardrails

## Goal

Expose a unified read-only search/status surface whose results tell non-programmers what is ready and what safe step remains.

## Tasks

1. Add inventory-derived readiness and strict protocol records.
2. Add `index workspace status/search` with all/project/skill/mcp selection, text output, and JSONL parity.
3. Add bounded read-only prompt augmentation for project/Skill/MCP discovery intent.
4. Prove the CLI exposes no install, authorize, execute, or start switches.
