<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Model JSON -> registry/preflight | Untrusted names, fields, paths, content, and process intent attempt to cross into effects. |
| Project filesystem -> read/write adapter | Symlinks/reparse points, binary/secrets, races, and conflicts can violate containment or integrity. |
| Tool action -> child process | Executable/argv/env/cwd/output/cancellation determine whether bounded diagnostics become arbitrary execution. |
| Approval UI -> invocation decision | Human intent must bind to the exact normalized call without manipulation or persistence. |
| Tool output -> journal/Provider/terminal | Secrets, raw stderr, control sequences, and oversized data must not escape. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-03-08 | Elevation of Privilege | path/preflight | critical | mitigate | canonical root/ancestor checks, protected metadata, symlink/reparse fixtures |
| T-03-09 | Information Disclosure | read/result | critical | mitigate | secret path/content detection, UTF-8/binary limits, bounded redacted envelope |
| T-03-10 | Tampering | patch/write | critical | mitigate | expected SHA-256/occurrence counts, in-memory validation, atomic same-dir persistence |
| T-03-11 | Elevation of Privilege | process adapter | critical | mitigate | finite action enum, shell=false, fixed argv/safe env, structural exclusion of interpreters/network/install |
| T-03-12 | Tampering | Git/npm | high | mitigate | fixed non-mutating Git operations and validated existing diagnostic scripts only |
| T-03-13 | Denial of Service | child/output | high | mitigate | timeout/combined-byte cap/cancellation/kill-and-wait/serial execution |
| T-03-14 | Spoofing | approval presentation | high | mitigate | normalized call receipt, exact call ID, explicit yes, no sticky grants |
| T-03-15 | Information Disclosure | terminal/journal | high | mitigate | sanitize controls, exclude stderr/secrets/raw reasoning, adversarial marker scans |
</threat_model>
