# Plan 07-02: Automatic Discovery and Behavioral Command Contracts

## Objective

Make the non-programmer project finder reachable from ordinary product use and replace parser-only parity with executable outcomes or explicit approved differences.

## Tasks

1. Embed the strict source-backed project catalog in the Rust binary and make `--catalog` an optional override.
2. Add bounded automatic discovery context for natural-language open-source/tool requests while preserving BM25-first, candidate-only semantic reranking and no installation side effect.
3. Wire retry and lifecycle-aware commands; define tested output contracts for provider/model/api/interrupt/trace cases that cannot safely mutate mid-turn.
4. Extend compatibility fixtures/harness to validate outcomes and approved differences, not only parsing.

## Verification

- `cargo test -p minimax-cli --locked --test discovery_commands`
- `cargo test -p minimax-retrieval --locked`
- `npm run verify:rust-contracts`

## Requirements

COMP-01, COMP-04, RUN-02, CLI-01, RETR-03
